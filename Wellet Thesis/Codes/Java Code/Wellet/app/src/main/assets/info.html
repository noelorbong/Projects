About

This app has been developed for the use on Tracking your wallet through bluetooth and GPS.

How to Use

First use of the app needs the bluetooth to be configured. Just Simply select the bluetooth option from the navigation menu.
Upon clicking the bluetooth option, a view of bluetooth information will appear as well as the button to change the bluetooth. Once the Change Bluettoth button selected, a list of Bluetooth paired devices will appear, then simply select the bluetooth of the wallet to get connected.
Note: The next time you open the app, it will automatically connect to the last buetooth device you selected.

The Map option in the navigation view shows the location of the wallet. Just make sure you are connected to the internet
